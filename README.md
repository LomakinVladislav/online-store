 ## Запуск backend'a

 1. Перейти в папку backend
 ```cd .\backend\```
 2. Перейти в витруальное окружение
 ```.\venv\Scripts\activate```
 3. Запустить файл src\main.py
 ```python .\src\main.py``


## Запуск frontend'a

 1. Перейти в папку frontend
 ```cd .\frontend\```
 2. Выполнить команду npm start
 ```npm start```
